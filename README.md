# Student Performance Analysis

## Project Overview
This project analyzes student performance using Python and Seaborn to understand the factors affecting student scores.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset Features
- Gender
- Ethnic Group
- Parent Education
- Parent Marital Status
- Weekly Study Hours
- Sports Participation
- Math Score
- Reading Score
- Writing Score

## Key Insights
- Gender distribution of students.
- Impact of parent education on scores.
- Parent marital status and student performance.
- Distribution of ethnic groups.
- Correlation between subjects.
- Study hours vs scores.
- Sports participation vs scores.
- Reading and writing score relationship.
- Missing value analysis.

## Visualizations
- Count Plot
- Heatmap
- Box Plot
- Pie Chart
- Scatter Plot
